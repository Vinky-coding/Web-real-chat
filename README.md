### Setup .env file

```js
MONGODB_URI=...
PORT=5002
JWT_SECRET=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
NODE_ENV=development
```

### build app

```shell
npm run build
```

### Hướng dẫn tải 

```shell
cd backend
npm install
```
```shell
cd frontend
npm install
```


### Chạy app
-> Lưu ý: chạy backend và frontend riêng lẻ.

```shell
npm run dev
```
